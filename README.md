# SAPNA — Sapna's Art Studio

Website for **Sapna's Art Studio**, an Ahmedabad-based handcrafted art business. Showcases macrame wall hangings, paintings, stitching art, and DIY kits — all inspired by travel and wanderlust.

## What It Does

A multi-page SPA that acts as an online storefront and portfolio for Sapna's handmade art products — allowing customers to browse collections, explore DIY kits, and connect with the studio.

## Key Features

- **Multi-route SPA** using React Router 6 — smooth page-to-page navigation
- **Product showcase** — macrame wall hangings, paintings, stitching art, DIY kits
- **Vite 5 build pipeline** — blazing fast development and production builds
- **Helper scripts** — `preview.sh` and `start.sh` for quick local setup
- **Custom domain** via CNAME (GitHub Pages hosted)
- **Static public assets** for images and media
- **Setup guide** (`SETUP_GUIDE.md`) for easy onboarding

## Tech Stack

| Technology | Purpose |
|------------|---------|
| React 18 | UI components |
| React Router DOM 6 | Client-side routing |
| Vite 5 | Build tool & dev server |
| GitHub Pages | Hosting |

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
./start.sh
# or: npm run dev

# Preview production build
./preview.sh
```

Art buyers, gift shoppers, and anyone looking for handcrafted home decor in Ahmedabad and beyond.
